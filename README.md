# ゲーム型授業シリーズ

児童福祉施設の小4〜6女子クラス向け「画面で体験するゲーム型授業」。全8作（理科6・社会2）。

各作品は外部依存なしの単一HTMLファイルで、ブラウザで開くだけで動作する。

## 作品一覧

| タイトル | 単元 | 教科 |
|---|---|---|
| 月よみ堂 | 月の満ち欠け | 理科小6 |
| まほう薬屋ほしずく | もののとけ方 | 理科小5 |
| ひだまり探偵社 | 地図の見方 | 社会小4 |
| おなかの探検隊 | 消化 | 理科小6 |
| 星めぐりの塔 | 星の動き | 理科小4 |
| みずの旅 | 水はどこから | 社会小4 |
| ながれる水の探検隊 | 流れる水のはたらき | 理科小5 |
| ちそう発掘隊 | 大地のつくり・地層 | 理科小6 |

`index.html` が一覧ページ。各作品の `◯◯_じゅぎょう画面.html` が授業本編、`◯◯_きろくシートA4.html` が印刷用の記録シート。
